Definitions

How to define a table?
Using HTML, you can define a table using the tag <table>. The size of your table is controlled by the amount of rows and columns that you create using <tr> tags and <td> tags.
When to use <tr>
The <tr> tag is used to create table rows in a table. The number of times you use this tags defines the amount of rows in your table.
When to use <th>
The table header tag, <th> is used instead of the table data tag <td> when that row is a header. It bolds the contents of each cell to indicate that it is a header.
When to use <td>
The <td> tag is used to define a standard data cell within a HTML table.
When to use <thead>
The <thead> tag is used in an HTML table to specify header content. This tag automatically bolds its content to indicate that it is a header.
When to use <tfoot>
The <tfoot> tag is used in an HTML table to specify footer content.
When to use <caption>
The <caption> tag is used with an HTML table as a title, to describe what the table is displaying. It is used directly following the <table> tag.
The difference between colspan and rowspan
Colspan spans the columns of a table, and merges the cells in a row together. Rowspan spans the rows of a table and merges column cells together.
The difference between cell spacing and padding
Cell spacing defines the amount of pixels between each cell in an HTML table. Cell padding defines how much space is between the cell content and the cell border in an HTML table.
How to define table borders using CSS
To define table borders using CSS you must first decide if you want to apply the styling to the table, the table header, and/or the table data. You can then style your table using properties such as "border-style", which allows you to make a border dotted, dashed, etc.
How to control border widths using CSS
You can apply the property "border-width" to a table, table header and/or table data to control the width of the borders.
